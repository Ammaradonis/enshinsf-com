# Enshin Karate SF

Static Cloudflare Pages site for Enshin Karate San Francisco, rebuilt from the JSON capture documents in this repository.

## Pages

- `index.html`
- `enshin.html`
- `about.html`
- `location.html`
- `schedule.html`
- `contact.html`

Cloudflare Pages can deploy this repository directly from `main` with no build command and `/` as the output directory.
